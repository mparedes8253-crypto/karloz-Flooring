# Karloz Flooring

## Overview
Karloz Flooring is a project aimed at providing a user-friendly platform for customers to explore various flooring options. It allows users to view different types of flooring products and helps in making informed decisions.

## Features
- Browse different flooring types and styles.
- Interactive product details for each flooring option.
- User-friendly interface designed for easy navigation.

## Setup Instructions
To get started with the Karloz Flooring project, follow these steps:

1. **Clone the Repository**  
   To clone the repository, run the following command:
   ```bash
   git clone https://github.com/mparedes8253-crypto/karloz-Flooring.git
   ```  

2. **Install Dependencies**  
   Navigate to the project directory and install the required dependencies:
   ```bash
   cd karloz-Flooring
   npm install
   ```  

3. **Run the Project Locally**  
   After installing the dependencies, start the development server:
   ```bash
   npm run dev
   ```  
   The project should now be running at `http://localhost:3000`.

## Deployment Guide to Vercel
To deploy the Karloz Flooring project on Vercel, follow these instructions:
1. Go to [Vercel](https://vercel.com/) and sign in.
2. Click on the "New Project" button.
3. Import your GitHub repository.
4. Configure the project settings as needed and click on "Deploy."
5. Your project will be deployed and accessible via a unique Vercel URL.

## Adding Portfolio Images

3. Make sure to update the relevant sections in the project files to include the new images.
